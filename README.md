# Whirlpool

A lightweight C++ framework for SDL3.

## Overview
Whirlpool is a simple and efficient framework designed to simplify SDL3 integration for C++ projects.

## Dependencies
- **SDL3**: Distributed under the [Zlib License](https://www.zlib.net/zlib_license.html).

## Features
- Simplified window management.
- Custom math utilities.
- Static build configuration for easier deployment.
